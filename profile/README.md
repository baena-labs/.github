# baena-labs

`baena-labs` is the home for our internal engineering projects, standards, and
automation.

## Repositories

- `org-config`: GitHub governance, rulesets, standards, and rollout automation
- application repositories: product and service code

## Working Principles

- pull requests before merge
- automated checks on default branches
- least privilege in GitHub Actions
- security scanning by default
- human ownership of production changes
