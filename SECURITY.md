# Security Policy

## Supported Repositories

Security issues may affect any repository in the `baena-labs` organization.

## Reporting a Vulnerability

Do not open public issues for suspected vulnerabilities.

Report security concerns privately via **GitHub Security Advisories** — open a
private advisory on the affected repository under _Security → Report a vulnerability_.

Include the following in your report:

- affected repository
- suspected impact
- reproduction steps
- proof-of-concept or logs, if available

If the report includes credentials or secrets, rotate them immediately after
sharing evidence.

## Response Expectations

- We will acknowledge receipt as soon as practical.
- We will validate, triage, and prioritize based on impact and exploitability.
- We may request additional evidence to reproduce the issue safely.

## Disclosure

Please allow reasonable time for remediation before public disclosure.
